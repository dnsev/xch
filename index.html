Firefox:
  1. Install Greasemonkey
  2. Install xch
Chrome: (currently untested)
  1. Download xch to your computer (save as...)
  2. Open chrome://extensions in your browser
  3. Drag and drop the xch.user.js file into the tab

xch is a userscript for 38chan.net designed to make board usage easier and have more features than the vanilla setup.
